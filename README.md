# blanja-api
Simple, easy implementation of the private web API.

## About The Project
API build in Express.js and mySQL for [yae48/blanja-web](https://github.com/yae48/blanja-web)

### Requirements
* npm
    [Node.js](https://nodejs.org/en/download/)
* express.js
    ```sh
  npm install express
  ```
* mysql
    ```sh
  npm install mysql
  ```
* morgan
express.js
    ```sh
  npm install morgan
  ```

### Usage
```sh
  node index
  ```
